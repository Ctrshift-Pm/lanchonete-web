# Lanchonete do Bairro

Projeto criado para praticar Git, GitHub e GitFlow.

Alteração feita diretamente no GitHub para testar git pull.

## Como usar

Abra o arquivo `index.html` no navegador para acessar a página principal.
Use o menu superior para navegar até o cardápio e a página de promoções.

## Estrutura

- `index.html`: página principal da lanchonete.
- `promocoes.html`: página com promoções do dia.
- `style.css`: estilos visuais do site.
- `cardapio.js`: dados e renderização do cardápio interativo.
