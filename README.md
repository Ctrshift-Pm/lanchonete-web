# Lanchonete do Bairro

Projeto criado para praticar Git, GitHub e GitFlow.

Alteracao feita diretamente no GitHub para testar git pull.

## Como usar

Abra o arquivo `index.html` no navegador para acessar a pagina principal.
Use o menu superior para navegar ate o cardapio e a pagina de promocoes.

## Estrutura

- `index.html`: pagina principal da lanchonete.
- `promocoes.html`: pagina com promocoes do dia.
- `style.css`: estilos visuais do site.
- `cardapio.js`: dados e renderizacao do cardapio interativo.
